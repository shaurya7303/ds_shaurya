This project analyzes the relationship between market sentiment and trader behavior using real trade execution data from Hyperliquid and the Bitcoin Fear & Greed Index.

The objective is to understand how sentiment influences: Profitability, Riskexposure Tradesizing ,Directional bias

Methodology

Clean and preprocess both datasets
Align data temporally using daily dates
Merge trades with sentiment labels
Engineer profitability and risk metrics
Perform sentiment-level and trader-level analysis
Visualize behavioral differences
Extract actionable insights
